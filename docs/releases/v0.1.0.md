# SynthHub 0.1.0

First alpha release of SynthHub, a dataframe-first Python library for
differentially private synthetic data.

## Highlights

- Unified `Synthesizer.fit/sample/evaluate` API.
- Public schema support for formal DP workflows.
- Live CI smoke coverage for:
  - DataSynthesizer PrivBayes and independent mode
  - Private-PGM AIM and MST with upstream mechanisms
  - SmartNoise MWEM, AIM, and MST
- Adapter contracts for SynthCity and SmartNoise GAN-family aliases.
- Utility and privacy evaluation reports with train-on-synthetic scoring and
  membership-inference audit metrics.
- Reproducible sklearn benchmark outputs.
- PyPI trusted-publishing workflow, package metadata checks, and wheel install
  smoke coverage.

## Install

Install from PyPI:

```bash
python -m pip install "synthhub[datasynthesizer]"
```

Install the current development version from GitHub:

```bash
python -m pip install "synthhub[datasynthesizer] @ git+https://github.com/tauptlab/synthhub.git"
```

## Release Checks

The release candidate has passed:

```text
python -m pytest -q
python -m build
python -m twine check dist/*
GitHub Actions CI
```

## Known Caveats

- SynthHub is alpha software.
- Formal DP guarantees require public preprocessing metadata. Use an explicit
  public `Schema` for formal workflows.
- Private-PGM AIM/MST require the upstream `mechanisms/` folder on
  `PYTHONPATH`.
- SynthCity and SmartNoise GAN-family adapters are contract-tested but not yet
  live-tested in CI.
